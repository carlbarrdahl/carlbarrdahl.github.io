# Marketplace for music creators

An increasing amount of music creators are using YouTube to teach music production. The projects featured in these videos contain valuable presets, samples and melodies in MIDI format. There is no easy way for the creators to monetize these project files.

Project is built using Next.js, GraphQL, PostgreSQL and deloyed to gcloud.
